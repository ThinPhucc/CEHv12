# Section 02: Hacking Methodology

## CEH Hacking Methodology
The CEH hacking methodology consists of the following phases:
- Gaining access
  - Cracking passwords
  - Vulnerability exploitation
- Escalating privileges
- Maintaining access
  - Executing applications
  - Hiding files
- Covering tracks
  - Clearing logs
